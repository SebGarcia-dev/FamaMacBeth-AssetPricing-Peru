# FamaMacBeth-AssetPricing-Peru
Este repositorio contiene la implementación empírica de modelos multifactoriales de Asset Pricing aplicados al mercado de valores peruano. Utilizando el procedimiento de regresión de dos etapas de Fama-MacBeth, se estima la significancia estadística y económica de diversas características empresariales (value, accruals, etc.) y factores de mercado.
